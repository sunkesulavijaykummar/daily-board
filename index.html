Excel backup & data
Your data lives in the cloud and syncs across every device you log in on. Export to Excel any time for a backup — or to feed the optional GitHub report runner.
How the GitHub live runner uses this file
  1. Click Export to Excel whenever you finish a day (or a week).
  2. Put study-log.xlsx into the data/ folder of your GitHub repo and commit it.
  3. The GitHub Action wakes up daily, reads the Excel file, and rebuilds your daily / weekly / monthly report pages automatically.
  4. Open your-name.github.io/daily-board/ to see this page, and /reports/ for auto-generated reports.
The Phase 2 kit (workflow + report generator + README) is delivered alongside this file — set it up once and reports build themselves.
